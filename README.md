# OOP Student Course Management System

## Lab Scope
A simple system built using Object-Oriented Programming (OOP) principles.

## Objective
This lab demonstrates how to:
- Create classes and objects
- Use constructors, fields, and properties
- Apply encapsulation, inheritance, and polymorphism
- Override methods
- Build reusable OOP code in C#

## Scenario
Build a system to manage:
- Students
- Courses
- Instructors
- Enrolments

## Requirements
- Create a base `Person` class  
- Create `Student` and `Instructor` classes (inherit from `Person`)  
- Create a `Course` class  
- Allow students to enrol in courses  
- Display student and course details  
- Use method overriding with `DisplayDetails()`
